# Halune

A modern AI Application Marketplace and Builder built with Next.js 14 (App Router), Tailwind CSS, and Lucide React.

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the app.

## Project Structure

- `app/` — Next.js App Router pages and layouts
- `components/` — Reusable UI components (Shadcn-style)
- `data/` — Mock data used for the marketplace
- `lib/` — Shared utilities
